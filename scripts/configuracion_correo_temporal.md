# Configuración de Correo Temporal para Pruebas

## Correo Temporal Recomendado

Para el envío exitoso de correos durante las pruebas, se recomienda utilizar el siguiente correo temporal:

**Correo:** `amaris-test-full@yopmail.com`

## Servicio de Correo Temporal

- **Servicio:** YOPmail
- **URL:** https://yopmail.com/es/
- **Descripción:** Correo electrónico temporal, anónimo y gratuito

## Configuración

1. Accede a https://yopmail.com/es/
2. Ingresa el correo temporal: `amaris-test-full`
3. Haz clic en el botón de verificación para acceder al buzón
4. El correo estará disponible y accesible sin necesidad de registro

## Nota

Este correo temporal es útil para:
- Pruebas de envío de notificaciones por email
- Validación de eventos de suscripción y cancelación
- Testing de integración con servicios de notificación
